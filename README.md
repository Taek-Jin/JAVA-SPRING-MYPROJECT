# 구현기능
<ul>
  <li>계정관련
    <ul>
      <li>회원가입, 로그인, 로그아웃</li>
      <li>이메일 저장</li>
      <li>MyPage(나의 정보, 장바구니, 내 QnA목록, 주문)</li>
    </ul>
  </li>
</ul>
<ul>
  <li>QnA게시판
    <ul>
      <li>QnA글 CRUD</li>
      <li>목록 페이징</li>
    </ul>
  </li>
</ul>
<ul>
  <li>장바구니
    <ul>
      <li>수량변경, 추가, 삭제</li>
      <li>개별주문, 전체주문</li>
    </ul>
  </li>
</ul>
<ul>
  <li>주문, 결제
    <ul>
      <li>기존배송지, 새로운배송지 등록</li>
      <li>카카오페이</li>
    </ul>
  </li>
</ul>
